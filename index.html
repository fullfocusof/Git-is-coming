Шпаргалка по Git

Команды Git
Команда Описание
git init Создание нового репозитория Git
git clone url Создание локальной копии репозитория Git, расположенной по указанному URL-адресу
git add file Добавление указанного файла в индекс Git
git add . Добавление всех измененных файлов в директории в индекс Git
git commit -m "comment" Создание нового коммита с комментарием
git commit -a -m "comment" Создание нового коммита с комментарием и автоматическим добавлением всех изменений в репозитории
git push url Отправка изменений из вашего локального репозитория в удаленный репозиторий по указанному URL-адресу
git status Вывод текущего состояния локального репозитория
git log Просмотр списка всех коммитов в локальном репозитории
git branch branch_name Создание новой ветки в локальном репозитории
git checkout branch_name Переключение на другую ветку в локальном репозитории
git checkout -b branch_name Создание новой ветки в репозитории и переключение на нее
git remote Список удаленных репозиториев, связанных с локальным
